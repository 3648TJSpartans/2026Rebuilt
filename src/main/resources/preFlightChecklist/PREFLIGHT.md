# Pre-Flight (or Match) Checklist
This is a list of things that you should probably be doing before a match starts to ensure everything actually works.

**Inspector:** ____________________

- [ ] Is the robot on?
- [ ] Is there a competition-ready battery in the robot?
- [ ] Are all of the status lights green?
  - [ ] Has the USB drive been replugged?
  - [ ] Can the robot currently see an AprilTag?
- [ ] Is everything in its starting configuration?
    - [ ] Turret
    - [ ] Hood
    - [ ] Climber
- [ ] Has the gyro been rezeroed?
- [ ] Have you been checking these without paying attention?
- [ ] Do we have the right auto selected for our:
  - [ ] Alliance
  - [ ] Auto strategy
